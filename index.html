Welcome to the download page for the NotAFan browser extension. Currently this extension is compatible with Mozilla FireFox, and can also be manually installed in Google Chrome. If you wish to install on Chrome please get in touch with me via FetLife.

NotAFan helps you to quickly identify user profiles on FetLife which contain links to external, commercially-oriented websites. This is done by inspecting links on the user profile page and cross-referencing them with known commercial websites. On detecting such a link, a pop-up is displayed which allows you to quickly block the user. The choice is still yours though! If you decide you want to proceed to viewing their profile just click the 'Cancel' button. NotAFan DOES NOT spider, crawl, or download data from FetLife.

If you have any issues with the plugin or questions please contact me on FetLife, and if you enjoy the plugin please join the NotAFan group to keep updated with any new developments.
